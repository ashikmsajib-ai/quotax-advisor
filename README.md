# QUOTAX ADVISOR - Professional Futures Signals

## Deploy to Railway (Recommended for Phone Access)

1. Push this folder to a new GitHub repository.
2. Go to https://railway.app
3. New Project → Deploy from GitHub repo
4. Add Environment Variables:
   - OPENROUTER_API_KEY = your_key
   - GEMINI_API_KEY (optional)
5. Deploy.

Your app will be live on a public URL — open it on your Android phone browser.

## Local Run
```bash
npm install
cp .env.example .env
npm run dev
```
